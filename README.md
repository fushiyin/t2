## Install:
> 
```
bun install
```
> If you use npm
```
npm install
npm ci 
```

## Run project:
* Development:
> 
```
bun dev
bun run dev
```
> If you use npm
```
npm run dev
``` 

## Expanding the ESLint configuration

## Format: Prettier
* Check:
> `bunx prettier . --check`
* Format:
> `bunx prettier . --write`
> or `bun format`

> You could change the name of the file to the "." dot in command to format specific file
